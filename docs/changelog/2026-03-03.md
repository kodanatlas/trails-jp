# 作業ログ 2026-03-03

## 実施内容

### 1. ScoreChart データソース変更
- エリート+年齢別の重複排除ロジックを「最大スコア採用」から「年齢別無差別クラスのみ採用」に変更
- `age_forest/無差別` + `age_sprint/S_無差別`（女子は `女子無差別` / `S_女子無差別`）のみ使用
- エリートランキングとの混在によるスコア不一致を根本的に解決

### 2. 比較機能に分布図追加
- `DistributionCharts.tsx`: `highlightAthletes` prop追加、複数選手のクロスヘア描画対応
- `CompareAthletes.tsx`: スコア推移チャートの下に `AthleteDistribution` を配置
- 選手A(#00e5ff) と選手B(#f97316) の位置を分布図上でハイライト表示

### 3. UI色の調整
- 成績レベル「好成績」: green-400 → cyan-400（水色）に変更
- 成績レベル「やや良い」: emerald-400 → green-400（緑）に変更
- 巡航速度チャートから `%` 単位を除去（LapCenter側に単位定義なし）

### 4. クラブ一覧の改善
- 50件表示制限を撤廃 → 全クラブ表示
- クラブ展開時に該当クラブカードが画面上部（80px余白）にスクロール
- 選手ページ遷移時に `history.pushState` で履歴記録 → ブラウザ戻るボタンでクラブ一覧に復帰
- クラブ→選手遷移時に `window.scrollTo({ top: 0 })` でページトップへ
- アクティブ人数の補足「半年以内に参加実績有り」をクラブ分布図の凡例に追加

### 5. スマホ表示の修正
- 全チャート親要素に `overflow-hidden` 追加（DistributionCharts, AthleteDetail, CompareAthletes）
- ページコンテナに `overflow-x-hidden` 追加

### 6. クラブ名寄せの拡充
- 日本語名の末尾数字除去ルール追加（青葉会18→青葉会、越王会'14→越王会 等）
- 個別エイリアス追加: 新潟→新潟大学、金沢→金沢大学、神戸→神戸大学
- 青葉会: 19名→62名に統合

### 7. ドキュメント更新
- `docs/analysis-system.md`: ScoreChartのデータソース変更、成績レベル色、クラブ名寄せルール、比較機能分布図、巡航速度単位、クラブ一覧改善を反映
